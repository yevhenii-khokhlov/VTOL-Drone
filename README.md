## Короткий опис проєкту

Проєкт присвячено розробці концепту промислового безпілотного літального апарату вертикального зльоту та посадки (БПЛА VTOL) класу fixed-wing hybrid для дистанційної інспекції сонячних електростанцій і трубопровідних магістралей. Апарат забезпечує автономний політ за маршрутом, цифрову передачу відео в реальному часі, навігацію з точністю RTK GNSS та сумісність з інфрачервоною камерою.
