# VTOL-Drone

## Проєкт
Промисловий VTOL для обстеження територій (сонячні панелі, трубопроводи)

## Мета проєкту
Розробити концепт безпілотної VTOL-платформи для інспекції довгих інженерних об’єктів із можливістю напівавтоматичного та автоматичного польоту за маршрутом.

## Основні вимоги
- Режим роботи: напівавтоматичний та автоматичний
- Час польоту: від 50 хвилин
- Дальність: до 15 км
- Передача відеосигналу: так
- Цифровий відеоканал: так
- RTK: так
- Підтримка інфрачервоної камери: так
- Захист від зіткнень: так
- Сенсори перешкод: так
- Висотомір відносно поверхні: так
- Транспортування в багажнику: так
- ПЗ для керування та обробки даних: так

## Структура репозиторію
- `README.md` — опис проєкту та навігація
- `docs/` — звіт, таблиці, текстові матеріали
- `cad/` — файли креслень із CAD-системи
- `diagrams/` — схеми з Draw.io
- `references/` — список джерел
- `images/` — ескізи, ілюстрації, рендери

## Плани розробки
### 1. Дослідження ринку
Підібрати мінімум 3 аналоги та порівняти їх за:
- масою
- часом польоту
- дальністю
- типом зв’язку
- наявністю RTK
- сумісністю з тепловізійною камерою
- ціною
- особливостями безпеки

### 2. Концепт платформи
Сформувати:
- тип VTOL-схеми
- компонування
- корисне навантаження
- електроніку
- енергетичну систему

### 3. CAD-розробка
Створити:
- загальний ескіз
- 3D-модель
- складальні креслення
- окремі деталі для друку / фрезерування

### 4. Документація
Підготувати:
- звіт Google Docs
- таблицю ТТХ
- перелік джерел
- схеми в Draw.io
