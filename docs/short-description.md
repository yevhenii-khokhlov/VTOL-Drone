# Короткий опис моделі

## 1. Назва та призначення

**Промисловий БПЛА VTOL для інспекції об'єктів інфраструктури**

Об'єктом проєктування є безпілотний літальний апарат (БПЛА) типу VTOL (Vertical Take-Off and Landing — апарат вертикального зльоту та посадки) у конфігурації fixed-wing hybrid. Апарат призначений для дистанційного обстеження протяжних промислових об'єктів: сонячних електростанцій і трубопровідних магістралей.

## 2. Область застосування

- Теплова інспекція фотоелектричних панелей — виявлення дефектних елементів і «гарячих точок» (hot-spot) засобами IR-тепловізійної зйомки;
- Відеоспостереження та картографування траси трубопроводів — контроль цілісності, виявлення витоків і механічних ушкоджень;
- Автономне патрулювання протяжних об'єктів за попередньо завантаженим маршрутом;
- Збирання геопросторових даних із точною геоприв'язкою на основі RTK GNSS.

## 3. Концепція платформи

Апарат виконано за схемою **fixed-wing VTOL** — поєднання несучого крила для ефективного крейсерського польоту та окремих підйомних роторів для вертикального зльоту та посадки (конфігурація tailsitter або quad-plane). Така конфігурація забезпечує:

- **Ефективність:** нижче питоме споживання енергії на одиницю дальності порівняно з мультикоптером за рахунок підйомної сили крила в горизонтальному польоті;
- **Мобільність при зльоті/посадці:** не потребує злітно-посадкової смуги, придатний для роботи у польових умовах;
- **Достатня автономність:** час польоту ≥ 50 хвилин, дальність до 15 км.

## 4. Основні технічні характеристики (проєктні вимоги)

| Параметр | Вимога |
|---|---|
| Схема VTOL | Fixed-wing hybrid (несуче крило + підйомні ротори) |
| Час польоту | ≥ 50 хв |
| Дальність | До 15 км |
| Навігація | RTK GNSS |
| Корисне навантаження | RGB-камера + IR-тепловізійна камера (модульно) |
| Захист від зіткнень | Датчики перешкод (передній/бічні напрямки) |
| Висотомір | Відносно поверхні (лідар або радарний) |
| Відеозв'язок | Цифровий |
| Польотні режими | Напівавтоматичний; автоматичний (waypoint) |
| Транспортування | У розібраному вигляді в багажнику легкового автомобіля |
| ПЗ | Керування польотом та обробка зібраних даних |

## 5. Склад систем

### 5.1 Планер
Фюзеляж із карбонових або скловолоконних композитів; модульні знімні крила для зручного транспортування. Конструкція орієнтована на складання без спеціального інструменту.

### 5.2 Силова установка
Електрична. Крейсерський тяговий двигун (задній або передній гвинт) + підйомні вертикальні ротори (quad-plane). Живлення — акумуляторна батарея з Li-Po або Li-Ion елементів.

### 5.3 Авіоніка та система керування
- Польотний контролер на базі ArduPlane / PX4 Autopilot (відкрите програмне забезпечення);
- RTK GNSS приймач для точного визначення координат;
- Телеметрія та захищений канал керування.

### 5.4 Корисне навантаження
- Модульна підвіска для RGB-відеокамери та IR-тепловізійної камери;
- Гіростабілізований підвіс (gimbal) для забезпечення якісної зйомки в польоті.

### 5.5 Система зв'язку
- Цифровий відеоканал (передача в реальному часі на наземну станцію);
- Телеметрія (двосторонній обмін службовими даними);
- Канал дистанційного керування.

### 5.6 Безпека
- Датчики перешкод — ультразвукові або оптичні сенсори для запобігання зіткненням;
- Висотомір відносно поверхні — лідарний або радарний модуль для функції terrain following;
- Функція автоматичного повернення додому (Return-to-Home) при втраті зв'язку або низькому заряді батареї.

### 5.7 Програмне забезпечення
- Наземна станція керування: Mission Planner або QGroundControl;
- Модуль планування маршруту інспекції;
- Модуль обробки та аналізу зібраних даних (термограм, відеозаписів, геопросторових знімків).

## 6. Відмінні особливості концепту

На відміну від більшості комерційних аналогів класу fixed-wing VTOL, пропонований концепт орієнтований на:

- **Відкриту архітектуру:** використання open-source автопілоту (ArduPilot / PX4) для незалежності від єдиного постачальника;
- **Максимальну сумісність:** підтримка широкого спектру IR-камер через стандартизовані інтерфейси;
- **Польову зручність:** швидке складання та розбирання, транспортування без спеціальних кейсів;
- **Доступність обслуговування:** можливість заміни ключових вузлів у польових умовах.
