# Структура звіту для Google Docs

> Цей документ описує рекомендовану структуру академічного звіту, який слід оформити у Google Docs відповідно до вимог університету. Додайте посилання на Google Docs у файл `README.md` (відкритий доступ до коментування).

---

## Структура звіту

### Титульна сторінка
- Назва університету, кафедри, спеціальності
- Тема проєкту: «Промисловий БПЛА VTOL для інспекції об'єктів інфраструктури»
- ПІБ студента, академічна група
- ПІБ керівника
- Місто та рік

---

### Завдання на проєкт
- Формулювання задачі (перелік вимог із ТЗ)
- Перелік обов'язкових результатів (ескіз, CAD, схеми, аналіз)
- Підпис керівника та студента

---

### Реферат
- Обсяг: ~½ сторінки
- Ключові слова: БПЛА, VTOL, fixed-wing, інспекція, RTK, IR-камера, автономний політ

### Перелік умовних скорочень

| Скорочення | Розшифровка |
|---|---|
| БПЛА | Безпілотний літальний апарат |
| VTOL | Vertical Take-Off and Landing (вертикальний зліт і посадка) |
| RTK | Real-Time Kinematic (кінематика в реальному часі) |
| IR | Infrared (інфрачервоний) |
| ТТХ | Тактико-технічні характеристики |
| ЗЛЗМ | Злітна маса (максимальна злітна маса) |
| ПЗ | Програмне забезпечення |
| GPS / GNSS | Global Navigation Satellite System |
| PPK | Post-Processing Kinematic |
| CAD | Computer-Aided Design |
| СЕС | Сонячна електростанція |
