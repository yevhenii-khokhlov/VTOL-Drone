# Специфікація основного обладнання

## 1. Призначення документа

Цей документ містить орієнтовну специфікацію основних бортових модулів для промислового fixed-wing VTOL-дрона, призначеного для інспекції сонячних електростанцій, трубопроводів та інших протяжних об’єктів інфраструктури.

## 2. Базові припущення про архітектуру

Для проєкту прийнято таку загальну конфігурацію:
- **тип апарата:** fixed-wing VTOL;
- **канал зв’язку:** Herelink (цифрове відео + телеметрія + керування);
- **автопілот:** окремий flight controller під ArduPilot або PX4;
- **додатковий комп’ютер:** Raspberry Pi 5;
- **корисне навантаження:** RGB-камера та інфрачервона камера;
- **силова установка:** 4 вертикальні мотори + 1 маршовий мотор;
- **живлення:** переважно 8S як основний варіант.

## 4. Орієнтовна маса та енергобаланс

Наведені нижче значення є інженерною оцінкою для попереднього проєктування та подальшого уточнення в CAD і під час підбору конкретних компонентів.

| Підсистема | Орієнтовна маса |
|---|---:|
| Базовий планер (фюзеляж, крило, оперення, кріплення) | 1.8–2.6 кг |
| 4 вертикальні мотори + кріплення + пропелери | 0.9–1.4 кг |
| Маршовий мотор + вузол встановлення | 0.25–0.45 кг |
| ESC та силова проводка | 0.35–0.7 кг |
| Акумулятор 8S | 1.5–3.5 кг |
| Автопілот + GPS/RTK + датчики | 0.25–0.5 кг |
| Raspberry Pi 5 + DC-DC | 0.15–0.25 кг |
| Herelink | 0.15–0.25 кг |
| RGB-камера | 0.2–0.6 кг |
| IR-камера | 0.15–0.5 кг |
| Сервоприводи та механіка керування | 0.15–0.35 кг |
| **Орієнтовна злітна маса** | **5.9–10.1 кг** |

За такої маси доцільно проєктувати силову установку із запасом тяги не менше ніж **1.8:1** відносно злітної маси для безпечного вертикального зльоту та стабільного переходу в горизонтальний політ.

## 5. Рекомендації щодо тяги та потужності

### 5.1 Вертикальні мотори
Для чотирьох вертикальних моторів доцільно закласти сумарну статичну тягу на рівні не менше **11–18 кг** залежно від остаточної маси апарата. За цього кожен мотор має забезпечувати приблизно **2.8–4.5 кг** статичної тяги у піковому режимі.

### 5.2 Маршовий мотор
Маршовий мотор повинен забезпечувати крейсерську тягу, достатню для польоту на економічному режимі без надмірного споживання енергії. Для цього його варто підбирати за гвинтом і крейсерською швидкістю після остаточного визначення аеродинаміки крила.

### 5.3 Акумулятор
Для першого працездатного прототипу рекомендується **8S LiPo**, оскільки він краще сприймає пікові струми під час вертикального зльоту. Якщо після стендових випробувань буде підтверджено допустимі струми, для пізнішої оптимізації часу польоту можна перейти до **8S Li-ion**.

## 6. Обґрунтування вибору 8S

Для заданої місії 8S є більш доцільним варіантом, ніж 6S, оскільки:
- зменшує робочі струми при тій самій потужності;
- знижує втрати в дротах і силових компонентах;
- забезпечує кращий запас по тязі;
- є більш ефективним для тривалого крейсерського режиму;
- краще підходить для важчої сенсорної та обчислювальної частини.

Водночас 6S може розглядатися як компромісний варіант для спрощення першого прототипу, однак для платформи з вимогами до дальності, часу польоту та корисного навантаження 8S є пріоритетним.

## 7. Функціональне призначення модулів

### 7.1 Herelink
Система Herelink використовується як основний цифровий канал зв’язку. Вона забезпечує передачу відеосигналу, телеметрії та команд керування, що відповідає вимогам до напівавтоматичного та автоматичного режимів польоту.

### 7.2 Raspberry Pi 5
Raspberry Pi 5 виконує роль додаткового бортового комп’ютера. Він не замінює автопілот, але забезпечує обробку відеопотоку, логіку місії, запис даних та можливу інтеграцію алгоритмів комп’ютерного зору.

### 7.3 Камери
Як основну RGB-камеру для проєкту доцільно використовувати **DJI Zenmuse X7** або камеру близького класу на базі **Sony Alpha / Sony RX0**, оскільки ці рішення є поширеними, мають добру якість зображення та застосовуються в аерозйомці й інспекційних місіях. Для теплового контролю рекомендується **DJI Zenmuse H20T** як універсальне рішення або **Kurbas-256 (Odd Systems)** як доступний локально орієнтований варіант, який може бути замовлений в Україні.

### 7.4 Мотори та сервоприводи
Вертикальні мотори забезпечують зліт і посадку, маршовий мотор — енергоефективний крейсерський політ. Сервоприводи використовуються для керування аеродинамічними поверхнями або допоміжними механізмами, якщо це передбачено компоновкою.

## 8. Висновок

Запропонована специфікація формує збалансовану апаратну основу для промислового fixed-wing VTOL-дрона інспекційного призначення. Поєднання Herelink, окремого автопілота, Raspberry Pi 5, тепловізійної та RGB-камери, а також живлення на базі 8S дозволяє забезпечити необхідні вимоги до дальності, часу польоту, безпеки та якості збирання даних.
