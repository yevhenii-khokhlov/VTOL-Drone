# Джерела та література

> У цей файл необхідно додавати всі використані джерела з активними посиланнями.

## Структура оформлення
1. Назва матеріалу.
2. Автор або організація.
3. Рік публікації.
4. Посилання.
5. Дата звернення (за потреби).
